# Security Policy

## Reporting a Vulnerability

If you discover a security issue in Lawyertec, please report it responsibly.

**Email:** security@lawyertec.mx

Please include:
- Description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

We aim to acknowledge reports within 48 hours.

## Scope

This policy covers:
- The Lawyertec landing page (`lawyertec-landing` repository)
- The `/api/waitlist` endpoint
- Supabase waitlist storage configuration

## Out of Scope

- Third-party services (Vercel, Supabase, OpenAI)
- Social engineering attacks
- Denial-of-service attacks

## Safe Harbor

We will not pursue legal action against researchers who follow responsible disclosure practices.
